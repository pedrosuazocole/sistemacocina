✨ INSTALACIÓN RÁPIDA:

1. Instalar Node.js desde: https://nodejs.org
   (Descargar la versión LTS - Long Term Support)

2. Abrir terminal en esta carpeta y ejecutar:
   npm install

3. Iniciar el sistema:
   npm start

4. Abrir navegador en: http://localhost:3000

5. Ingresar con:
   Usuario: admin
   Contraseña: admin123


📚 DOCUMENTACIÓN INCLUIDA:

- README.md           → Documentación técnica completa
- INSTALACION.md      → Guía paso a paso de instalación
- MANUAL_USUARIO.md   → Manual de uso del sistema

🚀 INICIO RÁPIDO (WINDOWS):

  1. Doble clic en: INSTALL.bat
  2. Esperar a que termine
  3. Doble clic en: START.bat


🚀 INICIO RÁPIDO (MAC/LINUX):

  1. Abrir terminal aquí
  2. Ejecutar: ./INSTALL.sh
  3. Ejecutar: ./START.sh

El sistema usa archivos JSON (carpeta /data)
